## 使用方式

### 编译使用

在`src`目录中`make`

```bash
./main -f ../test/test_1.txt -w 
cat solution.txt
rm solution.txt
```

**参数使用说明**

1. 参数的位置随意
2. 指定文件需要加上`-f`
3. `-w`   `-c`有且只有一个
4. `-h -t -n`与上面的参数可任意搭配

**输出的内容解释**

1. 只标示`-w`时，会输出单词链单词的数量，以及相应的单词链。
2. 只标示`-c`时，会输出单词链字符的数量，以及相应的单词链。
3. 标示了`-n`时，会输出满足要求的单词链的个数，然后分别输出相应的单词链，使用空行间隔。

### 清除

`make clean`
