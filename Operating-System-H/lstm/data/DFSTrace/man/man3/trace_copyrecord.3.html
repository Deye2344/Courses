NAME

Trace_CopyRecord -- Make a safe copy of a trace record

SYNOPSIS

void Trace_CopyRecord(sp, dpp)
dfs_header_t *sp;
dfs_header_t **dpp;

DESCRIPTION

Trace_CopyRecord makes a copy of the record pointed to by sp in *dpp . It allocates the storage for the record, and copies all the fields (including pathnames, not just the pointers).
