NAME

Trace_GetRefCount -- get reference count from trace record

SYNOPSIS

short Trace_GetRefCount(recPtr)
dfs_header_t *recPtr;

DESCRIPTION

Trace_GetRefCount returns the reference count of an object in CLOSE records. If the reference count field is not set, or if the record does not contain a reference count, the routine returns -1.
