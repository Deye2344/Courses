TRACE_FIDSEQUAL

Section: C Library Functions (3)     Updated: Feb 10, 1992    Return to Library's README

NAME

Trace_FidsEqual -- Compares fids

SYNOPSIS

int Trace_FidsEqual(fid1Ptr, fid2Ptr)
generic_fid_t *fid1Ptr;
generic_fid_t *fid2Ptr;

DESCRIPTION

Trace_FidsEqual compares two fids in pointers fid1Ptr and fid2Ptr and returns 1 if the fields match and 0 if they don't. Sending in NULL for either argument is tolerated.
