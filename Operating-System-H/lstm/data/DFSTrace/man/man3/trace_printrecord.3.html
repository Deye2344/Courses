NAME

Trace_PrintRecord,Trace_DumpRecord -- Print contents of trace record

SYNOPSIS

void Trace_PrintRecord(recPtr)
dfs_header_t *recPtr;
void Trace_DumpRecord(recPtr)
dfs_header_t *recPtr;

DESCRIPTION

These routines print the contents of a trace record recPtr to standard output. Trace_PrintRecord prints a one-line summary of the record. Trace_DumpRecord prints an exhaustive inventory of everything in the record.
