Section: C Library Functions (3)     Updated: Feb 10, 1992    Return to Library's README

NAME

Trace_PrintPreamble -- print the preamble of a trace

SYNOPSIS

int Trace_PrintPreamble(fp)
FILE *fp;

DESCRIPTION

Trace_PrintPreamble prints the preamble written for the trace represented by fp. Formats for the preamble vary, hence this routine.

DIAGNOSTICS

Returns TRACE_FILERECORDNOTFOUND if fp is not an open trace, otherwise returns TRACE_SUCCESS.
