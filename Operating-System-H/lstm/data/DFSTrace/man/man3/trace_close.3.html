NAME

Trace_Close -- close a trace file

SYNOPSIS

int Trace_Close(fp)
FILE *fp;

DESCRIPTION

Trace_Close closes the trace file and deletes the data structures associated with it.

DIAGNOSTICS

Returns TRACE_FILERECORDNOTFOUND if fp is not an open trace. Otherwise, returns TRACE_SUCCESS.
