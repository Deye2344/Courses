NAME

creplay -- execute commands from an untrace

SYNOPSIS

creplay file

DESCRIPTION

  creplay executes the commands in file produced by untrace. If the command file is a skeleton file, it will create a file system skeleton under the name ``root'' in the current directory. For a replay file, the action will happen under ``root'' in that directory.

OPTIONS

 

-g  Use if file system does not support chown of group ID.

-v  Verbose mode.

DIAGNOSTICS

If any of the commands in file does not complete successfully, creplay dies with an assertion failure. Any error will do, so make sure you have the appropriate permissions to write into the current directory, and make sure you have enough space for a (potentially) large tree.
