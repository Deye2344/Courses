Section: Maintenance Commands (8)     Updated: Feb 10, 1992    Return to Library's README

NAME

traceinfo -- print information on a tracing kernel

SYNOPSIS

traceinfo

DESCRIPTION

  Traceinfo prints a few statistics on what a tracing kernel has done. It indicates whether or not a tracing kernel is running, and if so, whether or not tracing is on, what level operations are being traced, and the number of bytes written by the kernel and read by an agent. Traceinfo should be run setuid root.

DIAGNOSTICS

Frequently encountered errors:

``cannot open /dev/dfstrace'' -- /dev/dfstrace was not installed, or traceinfo isn't being run as root.
