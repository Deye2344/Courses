NAME

tnote -- Annotate a trace

SYNOPSIS

tnote ``annotation''

DESCRIPTION

  tnote writes a NOTE trace record into the trace. A note record is simply a trace header followed by a character string annotation. The note should be enclosed in quotes.

DIAGNOSTICS

The following are frequently encountered errors:

``can't open /dev/dfstrace''. Either /dev/dfstrace doesn't exist, or the program isn't being run as root.
