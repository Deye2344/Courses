### 1.1 信息安全的概念

> 机密性、完整性、可用性也称为信息安全的三 要素。

信息安全属性

> 机密性、完整性、真实性、不可否认性、可靠性、可用性、可控性

信息安全威胁

> 信息泄露、非授权的篡改、拒绝服务、非法使用（非授权访问）、假冒、抵赖、网络与系统攻击、恶意代码、自然灾害、人为失误和故意破坏
